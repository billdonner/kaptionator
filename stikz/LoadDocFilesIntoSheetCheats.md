

##The Sheets

At its core, SheetCheats is a document repository for song content. Songs are stored as standard Document Types and organized by Song Title. The collection of Song Documents know to SheetCheats is known as the Corpus

![The Corpus](gm01.png)

##Loading and Unloading Document Files into Sheet Cheats

The corpus is loaded from external document files, include compressed zip files. Specific documents in the Corpus can be selectively removed from within the “Corpus Editor” feature of the app.

###File Types Supported

PNG,JPG,GIF,PDF,Pages,Word,Text are displayable

###Load from iTunes

You can load files thru iTunes File Sharing in the standard fashion. Connect your SheetCheats device to your computer, run iTunes, select your device, go to the Apps Page, go all the way to right hand side and scroll all the way down. 

Now look for the SheetCheats icon and select the app. Drag the files from your computer into iTunes and when all done, go hit the Import from iTunes function in the app.

###Load from iCloud

Select Import from iCloud, locate the  file you need to load, and double click. That’s it.

![A simple CSV file in Numbers](seven.png)

###Unloading

In the course of normal use you will sometimes need to remove files. You can use the Section Editing functions available from the Settings page.
